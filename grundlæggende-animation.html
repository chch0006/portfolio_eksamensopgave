Website - Spil

I dette tema blev vi introduceret til JavaScript og lærte at skabe interaktive og animerede elementer på websites. Vi arbejdede med CSS-animationer, spil-logik og UI-design, og kombinerede tekniske færdigheder med kreativt grafisk arbejde. Med Adobe Illustrator designede vi vores egne spilfigurer, baggrunde og UI-elementer. Opgaven var at udvikle et simpelt spil fra bunden – fra idé og grafik til funktionalitet i HTML, CSS og JavaScript. Derudover dokumenterede vi idéprocessen og udarbejdede aktivitets- og statemachine-diagrammer for spillets flow.

Besøg site

Løsning og proces

Vi blev introduceret til forskellige teknikker inden for skitsering og idéudvikling, samtidig med at vi arbejdede med frontend-teknologier som Adobe Illustrator, CSS og JavaScript. I JavaScript lærte vi om centrale begreber som funktioner, variabler og events, og i CSS arbejdede vi med positionering og animationer. Gennem kreative øvelser som papirprototyper og håndtegnede elementer udviklede jeg mit eget spil, Fairy Rescue, hvor jeg selv designede og byggede alle visuelle og tekniske elementer fra bunden.
